### 2.1 比赛中心

## 3. 数据爬虫模块

### 3.1 概述
- **位置**: `go_crawler/`
- **语言**: Go
- **数据源**: `vipc.cn` API
- **运行方式**: 独立运行，每次拉取当天数据

### 3.2 数据接口

| # | 接口名称 | URL | 说明 |
|---|---------|-----|------|
| 1 | 比赛列表 | `vipc.cn/i/live/football/date/today/next` | 今日全部比赛 |
| 2 | 历史战绩 | `vipc.cn/i/match/football/{matchId}/history` | 双方交锋记录 |
| 3 | 欧赔数据 | `vipc.cn/i/match/football/{matchId}/odds/euro` | 各公司欧赔 |
| 4 | 盘口数据 | `vipc.cn/i/match/football/{matchId}/odds/pankou` | 亚盘+大小球 |

### 3.3 入库表映射

| 接口 | DB 表 | 备注 |
|------|-------|------|
| 比赛列表 | `moneys` | 比赛基本信息（队名/比分/时间/联赛等） |
| 历史战绩 | `history_moneys` | JSON 存储（交锋/近况/联赛/排名） |
| 欧赔数据 | `odds_moneys` | JSON 存储（全部赔率 + 威廉/Bet365） |
| 盘口数据 | `pankou_moneys` | JSON 存储（亚盘/大小球 + Bet365） |

### 3.4 附加功能
- 自动下载球队 Logo → `public/footballimg/`
- 反爬策略: 随机 IP + User-Agent + 1.5 秒请求间隔
- 数据冲突处理: `ON CONFLICT UPDATE ALL`

## 4. 数据库设计

### 4.1 爬虫数据表

| 表名 | 说明 | 主要字段 |
|------|------|---------|
| `moneys` | 比赛主表 | match_id, date, league, home, guest, scores, logos, status |
| `history_moneys` | 历史战绩 | match_id, league_stat, against, recent, rank, future (JSON) |
| `odds_moneys` | 欧赔数据 | match_id, data(全部), avg_odds, william, bet365 (JSON) |
| `pankou_moneys` | 盘口数据 | match_id, asia_data, dxq_data, bet365 (JSON) |

### 4.2 业务数据表

| 表名 | 说明 | 主要字段 |
|------|------|---------|
| `users` | 用户 | username, nickname, avatar, email, badge, balance |
| `followed_teams` | 关注球队 | user_id, team_id, team_name, team_logo |
| `experts` | 专家 | name, avatar, specialty, accuracy, streak, followers, verified |
| `expert_tips` | 专家预测 | match_id, expert_id, prediction, confidence, analysis |
| `news` | 新闻 | title, summary, content, image_url, category, source, is_hot |
| `transfer_rumors` | 转会传闻 | player_name, from_club, to_club, value, trust_level, status |
| `notifications` | 通知 | user_id, type, title, message, icon, is_read |
| `wallet_transactions` | 交易记录 | user_id, type, amount, description, detail |
| `rewards` | 奖品 | name, description, icon, icon_color, cost, is_active |
| `search_histories` | 搜索历史 | user_id, query |
| `predictions` | 预测记录 | user_id, match_id, prediction, result, odds, amount, payout |
| `leaderboard_entries` | 排行榜 | user_id, username, accuracy, points, rank, period, change |
| `league_standings` | 积分榜 | league, team_name, rank, played, won, drawn, lost, points |
| `top_scorers` | 射手榜 | league, player_name, team_name, goals, assists, rank |

## 6. 可扩展功能建议

以下功能参考市面主流足球预测/体育 App（如 FotMob、SofaScore、365Scores、BetBuddy、Tipster）。

### 6.1 🔐 用户认证 (高优先级)

**当前缺失**: 无登录/注册/JWT 认证机制，所有 API 默认返回 userId=1。

| 功能 | 说明 |
|------|------|
| 注册/登录 | 邮箱+密码、手机号+验证码 |
| JWT Token 认证 | 中间件鉴权，所有用户相关 API 需 Token |
| 第三方登录 | Google/Apple/微信一键登录 |
| 忘记密码 | 邮箱验证码重置 |
| 用户资料编辑 | 修改头像/昵称/邮箱 |

### 6.2 📊 实时数据推送 (高优先级)

| 功能 | 说明 |
|------|------|
| WebSocket 实时比分 | 进球/红牌/中场等事件即时推送 |
| 比赛事件时间线 | 进球/换人/犯规/角球逐分钟记录 |
| 实时赔率变动 | 赔率波动推送 |
| 比赛状态变更 | 开赛/中场/结束通知 |

### 6.3 📈 高级数据分析

| 球队表现趋势图 | 近 N 场得失球/积分趋势 |
| 预期进球 (xG) | 高级统计指标展示 |
| 阵容对比 | 首发阵容可视化对比 |
